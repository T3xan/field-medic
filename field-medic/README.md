# ⛑ Field Medic — 365-Day Wilderness Medicine Course

A gamified, AI-powered daily field medicine curriculum built for hikers and campers.

## Features

- **365 lessons** across 10 modules (Foundations → Mastery)
- **AI-powered content** — each lesson is generated by Claude with trail-specific scenarios
- **Gamification** — XP, level system, daily streaks, 12 achievement badges
- **3-question quizzes** after every lesson with bonus XP
- **Field Notes** — save personal notes per lesson
- **Curriculum Map** — visual 365-day grid of all lessons
- **Fully offline-persistent** — all progress saved to localStorage

## Modules

1. 🏕️ Field Medicine Foundations (Days 1–30)
2. 🩹 Wounds & Bleeding Control (Days 31–60)
3. 🦴 Bones, Joints & Trauma (Days 61–90)
4. 🌡️ Environmental Emergencies (Days 91–130)
5. 🧭 Navigation & Rescue Signals (Days 131–160)
6. 🔥 Wilderness Survival Skills (Days 161–200)
7. 👧 Kid-Specific First Aid (Days 201–240)
8. 🐍 Plants, Animals & Toxins (Days 241–280)
9. 🫀 Body Systems & Illness (Days 281–320)
10. ⭐ Scenarios & Mastery (Days 321–365)

## Deploy to Vercel

1. Push this repo to GitHub
2. Import project at vercel.com
3. Framework: **Vite**
4. Build command: `npm run build`
5. Output directory: `dist`
6. Add environment variable: (none required — API key handled by Anthropic)

## Local Development

```bash
npm install
npm run dev
```
